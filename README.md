# rate.am ruble parser

You can get last ruble to amd price in Yerevan

### Installation

You need Gulp installed globally:
```sh
$ git clone https://github.com/slavikmanukyan/rateam-ruble.git
$ cd rateam-ruble
$ npm install
```
### Usage

```sh
$ npm start
```
